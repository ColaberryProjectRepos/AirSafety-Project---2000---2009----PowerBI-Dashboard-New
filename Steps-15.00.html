Step 7: Year Slicer 15

2)    In the format section: Do the following

Visual=Slicer Setting=Style = Between and Slider =ON with Color = Black

Visual=Slicer Setting=Slicer Heading = Off 

 

General=Properties Setting=Height = 98 and Width = 242; Position =Horizontal  = 1038 Vertical = 8   

 

General=Title Setting=Text  = “Year”  and Heading  = Heading 3; Font =DIN ; Size  = 15; Text Color = Black and Background Color = White with Horizontal Alignment = Middle

 

 
General=Effects Setting = On =Color  = “White”  and Transparency = 0; 

Visual border and Shadow = OFF; Header Icons = On; Background and Border = White; Icon = Black; Transparency = 0%
