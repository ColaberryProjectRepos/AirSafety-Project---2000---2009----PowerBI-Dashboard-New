Step 8: KPIs Cards (Left Side)

·         Total Accidents KPI Card
?

1)    Select the card from Visualization chart and drag Total Accidents from Accidents Measures table

2)    In the format section: Do the following

Visual=Callout value Setting=Font = Din and Size = 20 with Color = #118DFF

Display and Value places = Auto; Text and Source spaces = On

Category Label = Off

 

 

General=Properties Setting=Height = 105 and Width = 189; Position =Horizontal  = 0 Vertical = 161   

 

General=Title Setting=Text  = “Total Accidents”  and Heading  = Heading 3; Font =DIN ; Size  = 15; Text Color = White and Background Color = Black with Horizontal Alignment = Middle

 

 

General=Effects Setting = On =Color  = “White”  and Transparency = 100; 

Visual border and Shadow = OFF

 

General =Header Icons = On; Background and Border = White; Icon = Black; Transparency = 0%

 

·         Sub KPI Card (Accident_per_Month)

 

 

3)    Select the card from Visualization chart and drag Accident_per_Month from Accidents Measures table

4)    In the format section: Do the following

Visual=Callout value Setting=Font = Din and Size = 12 with Color = #252423

Display and Value places = Auto; Text and Source spaces = On

Category Label = Off

 

 

General=Properties Setting=Height = 29 and Width = 59; Position =Horizontal  = 0 Vertical = 237   

 

General=Title Setting; Header Icons; Tooltips =Off

 

 

General=Effects Setting = On =Color  = “White”  and Transparency = 100; 

Visual border and Shadow = OFF

 

 

·         Accident Button (Accident_per_Month)


5)    Select the Button from Insert Tab and choose the blank button

6)    In the format section: Do the following

Button=Shape Setting=Apple Setting to (State) =On hover

 

Shape=Speech Bubble; Round Corners = 0; Chat Bubble Height = 10%; Tail Position = Bottom Left; Tail angle = 90 

 

Button=Text Setting=Style=Text  = “Accidents per Month” and Font = Segoe UI and Size = 8 and Bold & Italic; Font Color = Black and Horizontal and Vertical Alignment = Left 

 

Button=Icon & Fill & Border & Shadow & Glow  Setting=Off

 

General=Properties Setting=Height = 67 and Width = 73; Position =Horizontal  = 1 Vertical = 199   

 

General=Title & Header Icons =Off
